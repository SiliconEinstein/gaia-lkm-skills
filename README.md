# gaia-lkm-skills — DEPRECATED

These skills have moved into **[`SiliconEinstein/Gaia`](https://github.com/SiliconEinstein/Gaia)**.
They now ship with the `gaia-lang` package and are used through the `gaia` CLI there
(`gaia skill register` to install them; `gaia search lkm` replaces the old LKM-search skill).

> **Note on versions/branches:** the bundled skill registry is a feature added in **v0.5** — use
> a v0.5+ build of Gaia (the trunk; the stable `release/0.4` line does not have it).

The old skill bodies remain in this repo's git history (tags up to `v2026.05.20.1`).
